# TDD Talk

Charla sobre Test Driven Development.

## Install

Asegurate de tener instalado [mdx-deck](https://github.com/jxnblk/mdx-deck) en tu 
PATH. Instala dependencias con `yarn` o `npm`.

## Run

`yarn start` levanta el dev server.
